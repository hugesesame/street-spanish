母音は5つ。常に同じ音

英語と違い、スペイン語の母音はどこに来ても同じ音。これが発音を覚えやすくする理由です。

A → ア
例: casa (カサ) = 家
E → エ
例: mesa (メサ) = テーブル
I → イ
例: vino (ビノ) = ワイン
O → オ
例: hotel (オテル) = ホテル
U → ウ
例: uva (ウバ) = ぶどう

特別な発音ルール8つ

H — 常に無音
「h」は読みません。完全に無音です。
hola → オラ（こんにちは） / hotel → オテル
J — ハ行（喉の奥から）
英語の「h」より強く、喉の奥から出す音です。
jamón → ハモン（ハム） / Juan → フアン
G — 2種類の音
e・i の前では「ヘ行」(=J と同じ音)、それ以外は「ガ行」。
general → ヘネラル / gato → ガト（猫）
C — 2種類の音
e・i の前では「ス」、それ以外は「カ行」。
cerveza → セルベサ（ビール） / café → カフェ
LL — ヤ行 / ジャ行
「ll」は「ヤ行」または「ジャ行」に近い音。
calle → カジェ（通り） / llama → ジャマ
Ñ — ニャ行
スペイン語固有の文字。「ニャ行」の音。
España → エスパーニャ / señor → セニョル
RR — 巻き舌
強い巻き舌音。語頭の「r」も同じく巻き舌。
arroz → アロス（米） / carro → カロ（車）
V = B（同じ音）
「v」と「b」はどちらも「バ行」の音で区別しません。
vino = bino → ビノ（ワイン）
アクセント記号について

á é í ó ú のように記号がついた母音はその音節を強く読みます。例：café（カフェ）、jamón（ハモン）

バル文化のルール

スペインのバルでは立ち飲みが普通です。カウンターの方が席より安いことも。バルは朝からオープンしており、朝食（コーヒーとパン）をとる場所でもあります。
